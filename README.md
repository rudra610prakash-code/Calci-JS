# 🧮 Calci-JS

A sleek, modern calculator built with vanilla JavaScript, HTML, and CSS. Featuring a beautiful glassmorphism design with smooth animations and intuitive controls.

---

## ✨ Features

- 🎯 **Clean Interface** - Minimalist design with a modern glassmorphism aesthetic
- ⚡ **Real-time Calculation** - Instant results as you type
- 🎨 **Dark Theme** - Easy on the eyes with a gradient dark background
- 📱 **Responsive Design** - Works seamlessly on different screen sizes
- 🎭 **Smooth Animations** - Interactive hover and click effects
- ⌨️ **Full Operations** - Addition, Subtraction, Multiplication, Division, and Percentage

---

## 🚀 Operations

| Button | Function |
|--------|----------|
| **AC** | All Clear - Reset calculator |
| **DEL** | Delete - Remove last digit |
| **%** | Percentage calculation |
| **/** | Division |
| **\*** | Multiplication |
| **-** | Subtraction |
| **+** | Addition |
| **.** | Decimal point |
| **=** | Calculate result |

---

## 🎮 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/rudra610prakash-code/Calci-JS.git
   cd Calci-JS
   ```

2. **Open the calculator**
   - Simply open `index.html` in your web browser
   - No installation or dependencies required!

3. **Start calculating**
   - Click buttons to enter numbers and operations
   - Press `=` to see the result
   - Use `AC` to clear all or `DEL` to undo last entry
